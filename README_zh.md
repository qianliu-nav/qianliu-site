## 🌟 项目简介

[**千流导航**](https://qianliu.com) (`https://qianliu.com`) 是一个专注于用户体验的轻量级网址导航网站。它摒弃繁杂装饰与广告干扰，以极简设计、清晰分类和快速加载为核心，帮助用户高效访问常用网站。

无论你是开发者、设计师、学生还是普通网民，千流导航都能为你提供一个干净、可靠的上网起点。

## ✨ 主要特点

- **极简界面**：无弹窗、无广告、无追踪，专注导航本身  
- **分类清晰**：涵盖「常用工具」「开发资源」「学习平台」「社交媒体」「生活服务」等高频场景  
- **响应式设计**：完美适配 PC、平板与手机端浏览  
- **快速加载**：静态页面 + CDN 加速，秒开无延迟  
- **持续更新**：站点列表由团队人工筛选维护，确保链接有效、内容优质

## 💡 为什么选择千流导航？

- 不收集用户隐私  
- 不植入推荐算法  
- 不跳转第三方短链  
- 只做一件事：**帮你更快找到想去的地方**
